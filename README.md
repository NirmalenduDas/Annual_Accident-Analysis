# Annual_Accident-Analysis
# 🚦 Traffic Accident Analysis and Prediction

## 📖 Project Overview

This project focuses on analyzing road accident data across different states in India using SQL and Python. The goal is to identify patterns, understand accident severity, and predict future accident trends using machine learning techniques. The project combines data analysis, visualization, and predictive modeling to derive meaningful insights.

---

## 🎯 Objectives

* Analyze accident trends across different states
* Identify high-risk regions based on severity index
* Perform SQL-based data analysis
* Visualize accident patterns using Python
* Build a machine learning model to predict future accidents

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* MySQL
* Matplotlib
* Power BI

---

## 📊 Dataset

The dataset contains information about road accidents across various Indian states.

### Features include:

* State
* Year
* Number of accidents
* Severity index
* Road fatality rate
* Railway accident data
